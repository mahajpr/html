Spicy Delight Restaurant Website

A simple HTML-only restaurant menu website created to practice semantic HTML structure, navigation links, lists, images, and page organization. This project showcases different food categories such as breakfast, lunch, dinner, drinks, and desserts.

Project Features : 
1.Navigation bar with menu links
2.Restaurant introduction section
3.Multiple food categories
4.Breakfast
5.Lunch
6.Dinner
7.Drinks
8.Desserts
9.Food descriptions and prices
10.Food images
11.Contact information footer

Technologies Used:
HTML5

HTML Concepts Practiced
1.Semantic HTML elements:
  <header>
  <nav>
  <section>
  <footer>
  <address>
2.Lists:
Ordered Lists <ol>
Unordered Lists <ul>
3.Images with <img>
4.Anchor links with <a>
5.Headings and paragraphs
